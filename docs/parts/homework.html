Learning C# with Xamarin: Inheritance and Polymorphism

Duration

30 minutes

Homework goals

Here you will write a console application to calculate the area of a shape. Sample output from the completed application looks like this:

You will need to perform the following steps:

  1. Create four classes: Shape, Rectangle, Square, Triangle.
    • The Shape class should be abstract and contain properties to store the Length and Width. The Shape class should also contain abstract methods to calculate the area, and describe itself as a string.
    • The Rectangle class should derive from Shape.
    • The Square class should derive from Rectangle.

      • Hint: The compiler will not force you to override any methods in this class because you are deriving from Rectangle which has already done that. However, maybe you should consider opting in.

    • The Triangle class should derive from Shape.
  2. Create a List of shapes in the Main method.
  3. Instantiate your List.
  4. Add some shapes to your List. Remember that you are using inheritance so you can add triangles, squares, etc.
  5. Use a foreach loop to iterate through the shapes List and print each shape to the console. You should also print the area of the shape by invoking the GetArea method.
  6. Try running the application. The reason why you are seeing class names is because the default string representation of a class is the class name. To solve this, override the ToString method in the Shape class only!
  7. Run the application again to see the correct results.

Solution

The provided Homework Resources/Completed folder contains a completed version of the program you can use to check your work. Your solution does not need to look exactly like this code, however the output should be correct as shown above.
