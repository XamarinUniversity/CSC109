Before you begin

This is a Xamarin cross-platform class about C#. To complete the coding exercises, you will need a Xamarin development environment (Mac or Windows) to run the exercise on. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

In this course, you will learn how to use inheritance in object-oriented programmng by applying generalization classes and creating a class that is derived from other. You will also know how to explain polymorphism, create virtual methods and override methods from derived classes.

Objectives

  1. Use inheritance to eliminate repeated code
  2. Use virtual methods and polymorphism to write generic code

Lecture

Learning C# with Xamarin: Inheritance and Polymorphism

Exercises

Exercise materials are available from GitHub. The code can be cloned through a desktop Git client or downloaded directly as a ZIP file.

  1. Inheritance: Create a derived class
    Exercise
    In this exercise, you will take an existing class which performs an Add operation and extract a base class from it to use with other types of mathematical operations. You will then create a few additional derived classes and use them in program.
  2. Polymorphism: Create and override virtual methods
    Exercise
    In this exercise you will create a list of polymorphic classes and iterate over the list calling an overridden virtual method.

Homework

Homework
This optional homework exercise will help to cement the topics in this class by having you write a C# application to calculate the area for a shape.
